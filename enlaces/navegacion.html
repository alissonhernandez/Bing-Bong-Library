Barra de navegación

Navega con Facilidad y Estilo! Nuestras barras de navegación han sido diseñadas pensando en ti, ofreciendo una experiencia intuitiva y fluida para explorar cada rincón de tu sitio web. Con un diseño adaptable y elegante, te acompañará sin importar el dispositivo que utilices.

Navegación Horizontal con Logo Visual

Experimenta una navegación familiar y centrada con nuestra barra horizontal, realzada por la presencia de un logo visual distintivo.


Te explicamos como funciona nuestro diseño:

<div class="navbar_furia_img"> <!--llama la clase SASS que define el diseño del navbar "furia" con imagen.--> <div class="logo" > <!--llama la clase SASS que define el diseño del logo <img src="../img/nav/furia.png" alt="logo"> <!--muestra una imagen como logo del navbar--> <div> <input type="checkbox" id="toggler"> <!--checkbox invisible que se usa para controlar la visibilidad del menú responsive--> <label for="toggler"> <!--label asociado al checkbox, al hacer clic en este label se activa/desactiva el checkbox--> <svg class="icon"> <!--contenedor para el icono SVG del menú--> <use xlink:href="../iconos/icons.svg#menu"></use> <!--utiliza un símbolo llamado "menu" </svg> </label> <div class="nav_items"> <!--contenedor para los elementos de navegación--> <a href="#" class="nav_link">Inicio</a> <!--enlace de navegación--> <a href="#" class="nav_link col">Documentación</a> <!--enlace de navegación--> <a href="#" class="nav_link col">Github</a> <!--enlace de navegación--> </div> </div>

Navegación Horizontal con Marca Textual

Sumérgete en una experiencia de navegación clara y concisa con nuestra barra vertical.

Te explicamos como funciona nuestro diseño:

<div class="navbar_alegria_nametext"> <!--llama la clase CSS 'navbar_alegria_nametext' que define el estilo del navbar con logo de texto "alegría"--> <div class="nav_marca">Alegria</div> <!--contenedor para el logo de texto del navbar, que en este caso es la palabra "Alegria"--> <ul class="nav_items"> <!--lista desordenada que contiene los elementos de navegación (enlaces)--> <li class="nav_link"> <a href="#">Inicio</a> </li> <!--elemento de la lista para el enlace "Inicio". La clase 'nav_link' aplica estilos individuales al enlace--> <li class="nav_link"> <a href="#">Servicios</a> </li> <!--elemento de la lista para el enlace "Servicios"--> <li class="nav_link"> <a href="#">Contacto</a> </li> <!--elemento de la lista para el enlace "Contacto"--> </ul> </div>
